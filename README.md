# Bank Marketing Classification
This repository contains a Jupyter Notebook comparing **Logistic Regression** and **Decision Tree** on the Bank Marketing dataset from UCI.  

### Notebook
- `BankMarketing_Classification_Samantha_Moataz.ipynb`: The main notebook with preprocessing, model training, evaluation, and visualizations.
- You can open and run the notebook directly in **Google Colab** using this link:
[Open in Colab](https://colab.research.google.com/github/SamanthaAValdez/bank-marketing-classification/blob/main/notebooks/BankMarketing_Classification_Samantha_Moataz.ipynb)

### How to run
- Open the notebook in **Google Colab** or **Jupyter Notebook**.
- Make sure `bank-full.csv` is in the same folder or upload it to Colab.

### Dataset
- [Bank Marketing Dataset - UCI ML Repository](https://archive-beta.ics.uci.edu/dataset/222/bank%2Bmarketing?utm_source=chatgpt.com)
